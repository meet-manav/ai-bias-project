# 🧠 AI Bias Detection System

## 🚀 About
This project detects bias in AI models used in areas like loan approval and hiring.

---

## 🎯 Problem
AI systems can sometimes make unfair decisions based on gender, age, or other factors.

---

## 💡 Solution
This project analyzes data and predictions to find bias and improve fairness.

---

## 🛠️ Tech Used
- Python  
- Pandas  
- Scikit-learn  
- Matplotlib  

---

## ▶️ How to Run

1. Download the project  
2. Install libraries:
   pip install -r requirements.txt  

3. Run:
   python main.py
